# Описание
WeePo -- это мобильное приложение для отслеживания важных для фотографов событий, таких как золотой и голубой час. Главные фунции приложения, отображение таймера до наступления важных событий и информирование о погоде

## Архитектура системы
Для описания архитектуры используеться инструмент Ice Panel. С диаграммой С4 можно ознакомиться по ссылке: https://s.icepanel.io/KyFv8zmGlyLj7s/7jFT
